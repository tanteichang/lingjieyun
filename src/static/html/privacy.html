隐私政策

一、引言

灵捷云平台（以下简称"我们"）非常重视用户的隐私保护，我们深知个人信息对您的重要性。本隐私政策旨在向您说明我们如何收集、使用、存储、保护和共享您的个人信息，以及您享有的权利。

请您务必仔细阅读本隐私政策，了解我们的信息处理实践。如果您不同意本隐私政策的任何内容，您应立即停止使用我们的服务。

二、我们收集的信息

我们收集的个人信息包括：

1. 您在注册账号时提供的信息，如姓名、手机号码、身份证号码等。

2. 您在使用我们的服务过程中产生的信息，如登录记录、操作记录等。

3. 您的设备信息，如设备型号、操作系统版本、IP地址等。

4. 您的位置信息，如GPS位置、IP地址等。

5. 其他您自愿提供的信息。

三、我们如何使用您的信息

我们使用您的信息主要是为了：

1. 向您提供服务，如账号管理、项目管理、信息查询等。

2. 改进我们的服务，如分析用户行为、优化服务体验等。

3. 保障服务的安全性，如防止欺诈、检测和处理违规行为等。

4. 向您发送通知，如服务更新、安全提醒等。

5. 其他与服务相关的目的。

四、我们如何存储您的信息

1. 我们会将您的信息存储在安全的服务器上，采取加密、访问控制等安全措施保护您的信息。

2. 我们会根据法律要求或业务需要，在合理的期限内存储您的信息。

3. 当您的信息不再需要时，我们会采取安全措施删除或匿名化处理您的信息。

五、我们如何共享您的信息

1. 我们不会向第三方出售您的个人信息。

2. 我们可能会与以下各方共享您的信息：

- 我们的关联公司，用于提供一致的服务。

- 我们的服务提供商，用于协助我们提供服务。

- 法律法规要求的政府机构或监管机构。

- 为了保护我们的合法权益或其他用户的合法权益。

六、您的权利

作为用户，您享有以下权利：

1. 访问您的个人信息。

2. 更正您的个人信息。

3. 删除您的个人信息。

4. 限制或拒绝我们处理您的个人信息。

5. 撤回您的同意。

6. 要求我们将您的信息转移给其他个人信息处理者。

7. 向我们提出投诉或建议。

七、Cookie和类似技术的使用

我们可能会使用Cookie和类似技术来收集和存储您的信息，以提供更好的服务体验。您可以通过浏览器设置拒绝Cookie，但这可能会影响您使用我们的部分服务。

八、未成年人保护

我们不向未成年人提供服务。如果我们发现未成年人使用我们的服务，我们会立即删除相关信息并终止服务。

九、隐私政策的变更

我们可能会根据业务发展或法律法规的变化更新本隐私政策。更新后的隐私政策将在我们的平台上公布，您可以随时查看。

如您继续使用我们的服务，则视为您接受更新后的隐私政策。
